# 🐚 Zsh en macOS

`zsh` es el shell estándar en macOS y será la base de trabajo en este entorno.

---

# 🎯 Objetivo

Tener un shell limpio, predecible y listo para uso diario.

---

# 📍 Archivo principal

La configuración de `zsh` vive en:

```bash id="m1a8kq"
~/.zshrc
```

---

# ⚙️ Configuración base

Incluye únicamente lo necesario:

* carga de alias
* variables de entorno necesarias
* configuración mínima del prompt

👉 Evita convertir `.zshrc` en un archivo grande y difícil de mantener.

---

# ✅ Recomendaciones

* Mantén una sola fuente de configuración
* Agrupa alias y ajustes de forma clara
* Documenta cualquier cambio no estándar

---

# ⚠️ Qué evitar

* plugins innecesarios al inicio
* configuraciones copiadas sin entenderlas
* múltiples archivos con lógica duplicada

---

# 🧠 Nota

El objetivo no es tener un shell “bonito”, sino un shell estable y fácil de usar.
