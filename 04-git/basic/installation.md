# 🔧 Instalación de Git

Este módulo cubre la instalación de Git en el sistema.

---

# 🎯 Objetivo

Tener Git instalado y disponible en la terminal.

---

# 🪟 Windows

## Opción recomendada

Instalar Git desde:

👉 https://git-scm.com

Durante la instalación:

* usar configuración por defecto
* permitir uso desde línea de comandos

---

## Verificación

```bash id="w3o2pl"
git --version
```

---

# 🍎 macOS

## Opción recomendada

Instalar herramientas de línea de comandos:

```bash id="z6j2qa"
xcode-select --install
```

Esto incluye Git.

---

## Verificación

```bash id="pqk8ru"
git --version
```

---

# ⚠️ Reglas

* Usar una sola instalación de Git
* Evitar múltiples versiones en el sistema
* No instalar Git mediante herramientas no documentadas

---

# ✅ Resultado esperado

* Git instalado
* Comando `git` disponible en terminal

---
